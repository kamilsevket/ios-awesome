# ``DesignSystemTypography``

A comprehensive typography system with Dynamic Type support for iOS applications.

## Overview

DesignSystemTypography provides a complete solution for managing typography in iOS applications. It includes:

- **Font Scales**: Predefined text sizes following Apple's Human Interface Guidelines
- **Dynamic Type**: Full support for accessibility text sizes
- **Custom Fonts**: Easy integration of custom font families
- **SwiftUI & UIKit**: Native support for both frameworks
