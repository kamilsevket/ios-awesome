# ``DesignSystem``

A comprehensive, modular design system for building consistent iOS applications with SwiftUI.

## Overview

DesignSystem provides a complete set of UI components, design tokens, and utilities for building beautiful, accessible, and consistent applications across Apple platforms.

### Key Features

- **Foundation Tokens**: Colors, typography, spacing, shadows, and icons
- **50+ UI Components**: Buttons, cards, forms, navigation, overlays, and more
- **Full Accessibility**: VoiceOver, Dynamic Type, and accessibility traits
- **Dark Mode**: Automatic light/dark mode support
- **Multi-Platform**: iOS, macOS, tvOS, and watchOS support
